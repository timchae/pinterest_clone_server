## 📖 프로젝트 소개

- 사용자가 스크랩하고 싶은 이미지를 포스팅하고, 다른 이용자와 공유하는 SNS인 Pinterest! 핀 이미지를 내가 만든 보드에 스크랩하고, 댓글로 아이디어를 나누며, 마음에 드는 댓글은 좋아요를 눌러 다른 사용자들과 소통하는 핀터레스트 주요 서비스를 Hinterest에서 만나보세요.
